# Route Studio Prompt

Design the `Route Studio` workspace for Prism as a route draft, explain, and publish workbench.

Goals:

- make route reasoning visible before publish
- support scenario simulation and blast-radius review
- connect route drafts to linked changes and investigations

Include:

- route draft editor
- matcher and policy review
- preview or explain panel
- scenario matrix
- blast radius summary
- publish linkage into Change Studio

Constraints:

- design for structured editing first, raw config second
- preserve room for compare mode and replay-based simulation
- make provider and auth dependencies explicit
